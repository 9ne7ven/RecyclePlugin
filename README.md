# ♻ RecyclePlugin

Advanced and fully configurable recycling system for Paper servers.

## Features

- GUI-based recycling
- Fully configurable rewards
- XP reward system (Economy support features incoming soon)
- Multi-language support
- Reload command
- Modern Adventure API text support

## Commands

/recycle - Open recycler  
/rpadmin reload - Reload configuration  

## Requirements

- Paper 1.20+
- Java 17+
